# 📦 Shell 执行器 v2.1.0 更新文档

> 从 `v2.0.0` 升级到 `v2.1.0` 的新特性一览

## 🆕 新增功能

### 1. 🛑 命令中断命令 `/shell stop`
- **用途**：立即中断当前正在执行的命令（相当于终端中按 `Ctrl+C`）
- **适用场景**：命令卡死、输出过多、误操作启动长时间任务
- **效果**：向当前 shell 会话发送 `SIGINT` 信号，命令被中断并返回已产生的输出

### 2. ⌨️ 控制键发送命令 `/ctrl <字母>`
- **用途**：向当前会话发送任意 `Ctrl+字母` 组合键
- **支持按键**：`c` (中断)、`d` (EOF/退出)、`z` (挂起)、`l` (清屏) 等
- **示例**：`/ctrl c` → 发送 `Ctrl+C`；`/ctrl d` → 发送 `Ctrl+D`（可退出当前 shell 进程）

### 3. ⏱️ 超时自动中断
- **行为优化**：当命令执行超过配置的 `超时时间` 时，插件会自动发送 `Ctrl+C` 中断命令，而非直接终止会话
- **返回信息**：超时后会附加 `[命令执行超时，已自动发送 Ctrl+C 中断]` 提示，以及中断后产生的输出

### 4. 🧹 插件卸载/重载时自动清理会话
- **新增方法** `terminate()`：当插件被禁用或重载时，自动关闭所有用户的 Shell 会话，释放系统资源

## 🔧 改进与优化

- 增强交互模式下的 `stop` 命令：即使处于等待输入状态，`/shell stop` 也会取消等待状态
- 控制键发送后读取即时输出，方便确认效果（如 `Ctrl+C` 后的中断信息）
- 日志信息更加详细，便于调试

## 📝 升级指南

### 通过 WebUI 升级（推荐）
1. 进入 AstrBot WebUI → **插件管理**
2. 找到 `shell执行器`，点击 **...**  ，然后点击**重新安装**
3. 无需重启 AstrBot，新功能立即生效

### 手动升级
1. 下载新版 `main.py`、`metadata.yaml`（版本号 `v2.1.0`）
2. 覆盖 `AstrBot/data/plugins/shell执行器/` 目录下的对应文件
3. 重启 AstrBot
